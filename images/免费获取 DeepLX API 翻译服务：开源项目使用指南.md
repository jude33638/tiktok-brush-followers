## 详细使用教程

### 第一步：获取 API 链接

1. 访问 DeepLX GitHub 项目
2. 授权 GitHub 账号
3. 获取专属的 DeepLX API Endpoint

### 第二步：配置翻译插件

1. 在沉浸式翻译插件中开启 Beta 模式
2. 找到 DeepLX(Beta) 设置项
3. 输入获取的 API URL
4. 测试服务连接是否成功
