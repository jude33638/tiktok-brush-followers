## 二、TradingView警报设置全流程

### 1. 创建有效交易警报
建议从以下几个维度设置警报条件：
- 价格突破关键支撑/阻力位
- 技术指标交叉（如MACD、均线系统）
- 超买超卖信号（RSI、KDJ等）

### 2. Webhook配置步骤
1. 点击图表界面"+"按钮创建新警报
2. 设置触发条件和频率参数
3. 在消息字段按规范格式填写订单详情：
json
{
  "symbol": "AAPL",
  "action": "buy",
  "quantity": 100,
  "price": 185.50
}

4. 输入第三方服务的Webhook接收地址

## 三、盈透证券TWS API对接方案

### 系统集成关键要点
- **运行环境**：需保持TWS客户端持续运行
- **网络要求**：建议使用固定IP的服务器部署
- **安全机制**：必须设置IP白名单和API权限控制

### 推荐第三方中间件选择标准
1. 支持REST API协议
2. 提供交易指令转换功能
3. 具备订单状态监控模块
4. 包含错误处理机制
5. 支持多账户管理
